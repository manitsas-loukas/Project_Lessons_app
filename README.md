# Project_Lessons_app
Η παρούσα εφαρμογή στοχεύει στη βελτίωση της επικοινωνίας μεταξύ καθηγητών, μαθητών/φοιτητών και γονέων, παρέχοντας ένα ενιαίο περιβάλλον παρακολούθησης της μαθησιακής διαδικασίας.  This application is designed to support effective communication and structured collaboration between teachers, students, and parents within an educational context.
